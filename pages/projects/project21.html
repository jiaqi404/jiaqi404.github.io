Textures Finder

AI-Driven Texture Search Tool

TexturesFinder is a tool for fast and accurate matching in texture databases using computer vision and similarity search algorithms. Users can input a local texture or randomly select a test image, set parameters, and search for similar textures within a dataset of 220 textures. Retrieved textures can be opened directly from their local paths for further operations. You can check my Github repo for further information. The link is on the right.
