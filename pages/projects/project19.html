Procedural Landscape Generation

Dynamic Stone Forest Terrain & Adjustable Pathways in UE5

This project utilizes UE5's Procedural Content Generation Framework to create a highly customizable terrain system through node-based generation blueprints.

The system adopts a layered generation architecture:

1. Base Layer: Terrain heightfield generation using noise maps,

2. Feature Layer: PCG Graph-driven rock clusters,

3. Interaction Layer: Spline-based dynamic path generation.
