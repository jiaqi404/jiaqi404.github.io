PCG Forest Scene

Procedurally Generated Wild Forest Scene

This project showcases a fully procedural forest scene built with Unreal Engine 5 and its Procedural Content Generation (PCG) framework. The workflow combines multiple tools:

1. Terrain Creation: Sculpted realistic landscapes using Gaea's terrain generation system

2. Material Mastery: Developed dynamic vegetation shaders leveraging UE's Landscape Layer Blend nodes and Landscape Grass Output for organic ground coverage

3. Procedural Ecosystem: Implemented PCG graphs to automatically populate forests with trees, vegetation and rocks.
